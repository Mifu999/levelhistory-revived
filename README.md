# Level History Revived

Look and play through deleted and outdated levels using [GDHistory](https://history.geometrydash.eu)!

Revived fork of [Level History](https://github.com/SergeyMC9730/levelhistory) by dogotrigger, updated for **GD 2.2081 / Geode 5.7.1** with bug fixes and new features (see `changelog.md`).

## Build

```bash
# requires the Geode SDK (GEODE_SDK env var)
geode build
```
