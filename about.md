# Level History Revived

Look through <cy>deleted and outdated levels</c> using <cp>GDHistory</c>!

Revived fork of <cg>Level History</c> by <cy>dogotrigger</c>, updated for GD 2.2081 / Geode 5.x with bug fixes and new features.

# How To Use

- Go to the <cy>Online</c> menu, then the <cy>search</c> screen
- Find the <cy>folder icon</c> in the filter menu
- Select the <cy>level provider</c> you want to use for this session
- Search by <cg>level ID</c> or by <cg>name</c>, browse every archived record with the arrows
- <cp>GG</c>

# Features

- Browse every <cy>archived record</c> of a level (version, date and source shown under each result)
- <cg>Play</c> old versions of levels directly in-game
- <cg>Save .gmd</c>: archive any record as a `.gmd` file (in the mod's save folder, `exports/`)
- <cg>History Page</c>: open the level's page on the GDHistory website
- <cy>Estimated upload date</c> for any level ID (based on GDHistory's date estimation API)
- Live <cy>archive statistics</c> in the provider info popup
- Configurable <cy>instance URL</c>, <cy>request timeout</c> and <cy>cache lifetime</c> in mod settings
